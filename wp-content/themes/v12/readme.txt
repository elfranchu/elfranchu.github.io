== Description ==
V12 is a simple responsive theme built on top of Bootstrap 3.0. The colors and fonts merge together to produce a high quality sleek theme.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in V12 in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel to provide an image logo.
